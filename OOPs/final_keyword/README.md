# final keyword
The final keyword is used to restrict modification. <br>
Once something is declared final, it cannot be changed.
- Used with variables, methods, and classes
- improve security and consistency.
- Helps create constat values.

### Properties
- final variable => value cannot change
- final method => cannot be @Overridden
- final class => cannot be inherited
- must be initialized once
- used for constants using static final
